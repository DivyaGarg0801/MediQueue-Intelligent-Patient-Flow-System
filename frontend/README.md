# Medique Hospital Management System

## Project Overview
A comprehensive hospital management system with Flask backend and React frontend.

## Features
- Patient Management
- Doctor Management  
- Appointment Scheduling
- Consultation Records
- Billing System
- Queue Management
- Dashboard Analytics

## Tech Stack
- Backend: Flask, MySQL
- Frontend: React, CSS3
- Database: MySQL 8.0
